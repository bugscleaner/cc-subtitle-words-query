# CC字幕 | 查词翻译 ｜谷歌翻译｜沙拉查词｜油猴脚本
在B站和YouTube学英语的时候，有时候开启了CC字幕，但是CC字幕没办法复制某个单词，得需要手动输入到翻译工具里，感觉有点麻烦，所以我做了一个油猴小插件，让CC字幕可以能够被选中复制，结合谷歌翻译或者沙拉查词就能很快的找到对应单词或句子的翻译了。
目前支持Bilibili、Youtube，其它使用CC字幕的网站原理大致一样。
